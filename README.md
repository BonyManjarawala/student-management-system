# Student-management-system(MongoDB)

# SCHEMA:

STUDENT

* id: Number,
* name: String,
* age: Number,
* cource: String

# GET, POST, PUT, DELETE REQUESTS:

ADD
* add new Student

RETRIEVE
* fetch student details based on student name

UPDATE
* update student (add/remove student)

DELETE
* delete student
